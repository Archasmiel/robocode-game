Sprite Renderer в Unity — це компонент, який використовується для відображення двовимірних графічних елементів (спрайтів) у 2D-іграх. Він відповідає за те, як виглядає спрайт на екрані.

Основні властивості Sprite Renderer:

Sprite
└ Задає зображення (текстуру), яке буде відображатися як спрайт.
└ Зображення потрібно імпортувати у форматі Sprite (2D and UI).


Color
└ Визначає колір або прозорість спрайта.
└ Використовується для створення ефектів, наприклад, тіней чи відтінків.


Material
└ Вказує матеріал для рендерингу спрайта.
└ Матеріал впливає на вигляд спрайта за допомогою шейдерів і текстур (наприклад, блиск чи світіння).


Flip X / Flip Y
└ Flip X перевертає спрайт горизонтально.
└ Flip Y перевертає спрайт вертикально.


Draw Mode
└ Визначає, як буде відображатися спрайт:
• Simple — стандартне відображення.
• Sliced — використовується для масштабування в стилі 9-slice.
• Tiled — текстура повторюється (тайлінг).


Size
└ Використовується в режимі Tiled для встановлення розміру тайлів.


Mask Interaction
└ Налаштовує, як спрайт взаємодіє з масками:
• Не взаємодіє.
• Відображається тільки в межах маски.
• Відображається за межами маски.


Additional Settings
Містить додаткові параметри залежно від шейдера чи особливостей об’єкта.	

└ Sorting Layer
  └ Визначає шар рендерингу, на якому знаходиться спрайт.
  └ Спрайти з вищим шаром відображаються поверх тих, що на нижчих шарах.

└ Order in Layer
  └ Визначає порядок відображення всередині одного шару.
  └ Спрайт із більшим значенням буде ближче до камери.
